To add YouTube or/and Vimeo videos in sidebar...

== Description ==

How to embed YouTube or Vimeo video in sidebar? Very easy. Just copy and paste Youtube or Vimeo url into widget.  

And that is it. If you want more than one video in sidebar, just use more than one widget.

== Installation ==

1. Upload the folder `side-video-embed` to the directory `/wp-content/plugins/`.
2. Activate the plugin via the 'Plugins' menu in your WordPress admin panel.
3. You can adjust the necessary settings in your WordPress widget area.
